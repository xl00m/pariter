### [ НАСТРОЙКА СВЯЗИ ]

В процессе установки Мастер спросит у тебя координаты:

*   **Домен** - адрес вашего дома в сети (например: `pariter.ru`)
*   **Email администратора** - на всякий случай
*   **Логин и пароль** - ключи от дверей, которые откроются только вам

### [ ПЕРВЫЙ КОНТАКТ ]

1.  Открой браузер.
2.  Зарегистрируй главного в команде или войди как администратор.
3.  Перейди в раздел **Спутники**.
4.  Создай приглашение. Отправь его тому, кто ждет.

С этого момента вы больше не одни в пучине хаоса.

---

### [ ФУНДАМЕНТ ]

Для существования вашего мира потребуется совсем немного кремния:

*   **Сервер:** VPS (Ubuntu/Debian)
*   **Память:** 512 MB RAM
*   **Пространство:** 1 GB диска
*   **Сеть:** Домен, смотрящий в душу сервера (на его IP)

Для ИИ какой-нибудь API.

### [ СОХРАНЕНИЕ ПАМЯТИ ]

В мире DELETE мы выбираем REMEMBER.
Вся ваша история. Каждое слово. Каждый инсайт.
Все хранится в одном файле. Береги его.

`>> /opt/pariter/pariter.db`
